# 🎬 StreamWave Analytics: Viewer Behaviour & Genre Performance

**Analysis of viewer behaviour and genre performance on a streaming platform using Excel. Built interactive dashboards to evaluate engagement, retention, and repeat viewing, supporting data-driven content investment decisions.**

---

## 📌 Overview
This project explores viewer activity on a streaming platform (StreamWave Entertainment) to understand which content genres drive engagement, retention, and repeat viewing.

With rising content costs and increasing competition, the goal was to support data-driven decisions on content investment and strategy.

---

## 🎯 Problem Statement
StreamWave operates a large and diverse content library, but:

- Viewer engagement is spread across multiple genres  
- Content production and licensing costs are increasing  
- Some users churn due to weak content alignment  

**The challenge:**  
👉 Identify the genres that truly sustain engagement and subscriber loyalty.

---

## 🧠 Objectives
- Analyse viewing behaviour across genres  
- Measure engagement (completion rate, watch patterns)  
- Identify repeat viewing behaviour  
- Evaluate retention impact by genre  
- Highlight underperforming categories  
- Provide clear recommendations for content strategy  

---

## 🛠️ Tools & Techniques
- Microsoft Excel  
  - Power Query (data cleaning & transformation)  
  - Pivot Tables (aggregation & analysis)  
  - Calculated fields (KPIs)  
  - Interactive Dashboard (slicers & visuals)  

---

## 📊 Dataset
The dataset includes:

- User data (User_ID, Age, Gender, Subscription details)  
- Viewing activity (Watch duration, Completion flag, Watch date)  
- Content metadata (Genre, Title, Release year, Duration)  

---

## 🔄 Approach

### 1. Data Preparation
- Imported and cleaned multiple CSV files  
- Removed duplicates and handled missing values  
- Standardised genre categories  

### 2. Analysis
Built pivot tables to analyse:
- Genre popularity  
- Completion rates  
- Repeat viewing  
- Retention vs churn  

### 3. KPI Development
- Engagement score (combined metrics)  
- Repeat viewing rate  
- Completion rate by genre  
- Retention impact  

### 4. Dashboard
- Designed an interactive dashboard with slicers  
- Enabled dynamic filtering by genre  
- Visualised trends and performance metrics  

---

## 📈 Key Insights
- **Drama (22.13%) and Comedy (21.40%)** are the most popular genres  
- **Repeat viewing is highest in Drama (55%) and Comedy (52%)**, indicating stronger loyalty  
- **Biography and Musical show 100% completion**, but very low demand (<0.2%)  
- **Horror and Musical underperform** in both engagement and repeat viewing  
- Retention is driven more by **repeat engagement than total views**  

---

## 💡 Recommendations
- Prioritise investment in **Drama and Comedy**  
- Reassess spend on **low-performing genres (e.g. Horror, Musical)**  
- Focus on **repeat viewing and completion rates** as core KPIs  
- Use genre insights to improve **recommendation systems and targeting**  

---

## 🚀 Business Impact
This project demonstrates how analysing viewer behaviour can:

- Improve content investment decisions  
- Reduce churn  
- Strengthen user engagement  
- Support long-term growth strategy
